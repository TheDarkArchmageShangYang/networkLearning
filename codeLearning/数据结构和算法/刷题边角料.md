## 2.交换两个元素的值

方法1

```c++
int temp = x;
x = y;
y = temp;
```

方法2

```c++
x ^= y; // x = x ^ y
y ^= x; // y = y ^ x = y ^ (x ^ y) = (y ^ y) ^ x = x
x ^= y; // x = x ^ y = x ^ y ^ x = y
```
