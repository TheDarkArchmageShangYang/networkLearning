# MarkDown语法

## 1.标题&目录

### 1.1标题

- MarkDown标题共有**六级**,和HTML相同

- 标题格式

  - # x 标题级数 \+ 空格 + 文本内容
  - 快捷键: Ctrl + 1/2/3/4/5/6

  ```markdown
  这是一段普通的文本
  
  # 这是一级标题
  ## 这是二级标题
  ### 这是三级标题
  #### 这是四级标题
  ##### 这是五级标题
  ###### 这是六级标题 
  ```

## 2.斜体/粗体

### 2.1斜体

- 斜体的格式:

  - \* \+ 文本内容 + \*
  - \_ + 文本内容 + \_(这里是1个\_)

  ```markdown
  这是一段普通文本
  
  *这是一段斜体文本*
  _这是一段斜体文件_
  ```

  

### 2.2粗体

- 粗体的格式:

  - \*\* + 文本内容 + \*\*

  - \_\_ + 文本内容 + \_\_(这里是2个\_)

```markdown
这是一段普通文本

**这是一段加粗文本**
__这是一段加粗文本__
```

### 2.3粗斜体

- 粗斜体的格式:

  - \*\*\* + 文本内容 + \*\*\*
  - \_\_\_ + 文本内容 + \_\_\_(这里是3个\_)
  - \*\*\_ + 文本内容 + \*\*\_
  - \_\*\* + 文本内容 + \_\*\*
  - \*\_\_ + 文本内容 + \*\_\_(这里是2个\_)
  - \_\_\* + 文本内容 + \_\_\*(这里是2个\_)

- 说明:

  - 粗斜体文本,首尾各有3个标识符

  ```markdown
  这是一段普通文本
  
  ***粗斜体文本1***
  ___粗斜体文本2___
  **_粗斜体文本3_**
  _**粗斜体文本4**_
  *__粗斜体文本5*__
  __*粗斜体文本6__*
  ```




## 3.线

### 3.1水平分割线

- 水平分割线的格式:

  - \-\-\-(这里是3个\-)

  ```markdown
  下面是一条水平分割线：
  ---
  ***
  ```

  

### 3.2文本删除线

- 删除线的格式:

  - \~\~ + 文本内容 + \~\~

  ```markdown
  ~~这是一段加了删除线的文本~~
  ```

  

### 3.3文本下划线

- 下划线的格式:

  - \<u> + 文本内容 + </u>

  ```markdown
  <u>这是一段加了下划线的文本</u>
  ```



## 4列表&引用

### 4.1有序列表

- 有序列表的格式:

  - 1\. \+ 空格 + 文本内容

- 说明:

  - 输入文本内容后,敲击 <strong><kbd>Enter</kbd></strong> 自动补全格式,并进入下个有序列表
  - 在同一个列表中,需要换行显示时,敲击 <strong><kbd>Shift</kbd></strong> \+ <strong><kbd>Enter</kbd></strong>,即可另起一行

  ```markdown
  1. 这是第一个有序列表 <!-- (Enter) -->
  2. 这是第二个有序列表 <!-- (Enter) -->
  3. 这是第三个有序列表 
  
  
  1. 这是第一个有序列表 <!-- (Shift + Enter) -->
     这是同个列表下，另起一行的文本内容 <!-- (Enter) -->
  2. 这是第二个有序列表 <!-- (Shift + Enter) -->
     这是同个列表下，另起一行的文本内容
  ```



### 4.2无序列表

- 无序列表的格式:

  - \- \+ 空格 + 文本内容

- 说明:

  - 输入文本内容后,敲击 <strong><kbd>Enter</kbd></strong> 自动补全格式,并进入下个无序列表
  - 在同一个列表中,需要换行显示时,敲击 <strong><kbd>Shift</kbd></strong> \+ <strong><kbd>Enter</kbd></strong>,即可另起一行

  ```markdown
  - 这是第1个无序列表 <!-- (Enter) -->
  - 这是第2个无序列表 <!-- (Enter) -->
  - 这是第3个无序列表
  
  - 这是第一个无序列表 <!-- (Shift + Enter) -->
    这是同个列表下，另起一行的文本内容
  - 这是第二个无序列表 <!-- (Shift + Enter) -->
    这是同个列表下，另起一行的文本内容
  ```

  

### 4.3引用

- 引用的格式:

  - \> + 文本内容

- 说明:

  - 同个引用段落内的换行敲击 <strong><kbd>Enter</kbd></strong> 即可
  - 若要添加第二个独立引用段落,连续敲击两下 <strong><kbd>Enter</kbd></strong> 即可

  ```markdown
  >这是第一段引用文本的第1行 <!-- (Enter) -->
  >这是第一段引用文本的第2行 <!-- (Enter) -->
  <!-- (Enter) -->
  >这是第二段引用文本的第1行 <!-- (Enter) -->
  >这是第二段引用文本内第2行
  ```

  >这是第一段引用文本的第1行 
  >这是第一段引用文本的第2行 

  >这是第二段引用文本的第1行
  >这是第二段引用文本内第2行

### 4.4缩进&退格

- 有序/无序列表的缩进与退格
  - 缩进的格式:
    - <strong><kbd>Tab</kbd></strong>
    - <strong><kbd>Ctrl</kbd></strong> \+ <strong><kbd>[</kbd></strong>
  - 退格的格式:
    - <strong><kbd>Shift</kbd></strong> \+ <strong><kbd>Tab</kbd></strong>
    - <strong><kbd>Ctrl</kbd></strong> \+ <strong><kbd>]</kbd></strong>
- 引用的缩进与退格
  - 缩进的格式:
    - 额外的<strong><kbd>></kbd></strong>
  - 退格的格式
    - 与列表相同



## 5.网页链接&图片

### 5.1网页链接

- 网页链接的格式:

  - \[ \+ 显示文本 \+ \] + \( + 链接地址 + 空格 + \" + 提示信息 + " + )

- 说明:

  - 显示文本是在渲染界面实际可见的文本,用以说明链接
  - 提示信息需要鼠标悬停于显示文本才能触发,这是可选项
  - 通常情况下需要按住<strong><kbd>Ctrl</kbd></strong>+鼠标左键点击才能跳转链接

  ```markdown
  [百度一下，你就知道](https://www.baidu.com "按住Ctrl点击跳转百度")
  ```

  [百度一下，你就知道](https://www.baidu.com "按住Ctrl点击跳转百度")



### 5.2图片

- 图片的格式:
  - \[ \+ 显示文本 \+ \] + \( + 链接地址 + 空格 + \" + 提示信息 + " + )
- 说明:
  - 显示文本在渲染界面不可见,没什么作用
  - 提示信息需要鼠标悬停于显示文本才能触发,这是可选项
  - 通常情况下需要按住<strong><kbd>Ctrl</kbd></strong>+鼠标左键点击才能跳转链接
- 调整图片大小的格式:
  - Typora可以右键缩放
  - 可以添加`style="zoom:50%;"`,数值可以自己修改



## 6.表格

- <strong><kbd>|</kbd></strong>是组成表格的主要框架

- <strong><kbd>-</kbd></strong>区分表头和表格

- <strong><kbd>:</kbd></strong>控制表格内文本内容的对齐方式

- Typora输入<strong><kbd>Ctrl</kbd></strong> \+ <strong><kbd>T</kbd></strong>快速插入表格

  ```markdown
  |这里是表头1|这里是表头2|这里是表头3|
  <!--区分表头和表格主体，:代表文本对齐方式，分别是左对齐，居中对齐，右对齐-->
  <!--不指定对齐方式时可以使用 |-|-| 区分表头和表格-->
  |:-|:-:|-:|
  |单元格数据1|单元格数据2|单元格数据3|
  |单元格数据4|单元格数据5|单元格数据6|
  
  |这里是表头1|这里是表头2|这里是表头3|
  |-|-|-|
  |单元格数据1|单元格数据2|单元格数据3|
  |单元格数据4|单元格数据5|单元格数据6|
  ```

  指定对齐方式:

  | 这里是表头1 | 这里是表头2 | 这里是表头3 |
  | :---------- | :---------: | ----------: |
  | 单元格数据1 | 单元格数据2 | 单元格数据3 |
  | 单元格数据4 | 单元格数据5 | 单元格数据6 |

  不指定对齐方式:

  | 这里是表头1 | 这里是表头2 | 这里是表头3 |
  | ----------- | ----------- | ----------- |
  | 单元格数据1 | 单元格数据2 | 单元格数据3 |
  | 单元格数据4 | 单元格数据5 | 单元格数据6 |



### 6.1表格内文本的换行

- Mardown中表格，它的宽高会根据单元格数据内的文本内容变化

- 当我们输入一段很长很长的文本，它所在的单元格会变得过宽

  ```markdown
  | 表头1 |  表头2 |
  |:-:|:-:|
  |这是第一行文本<br>这是另起一行的文本|普通文本|
  ```

  |                表头1                 |  表头2   |
  | :----------------------------------: | :------: |
  | 这是第一行文本<br>这是另起一行的文本 | 普通文本 |



## 7.代码域

### 7.1行内代码

- 行内代码的格式:

  - \` + 代码 + \`

- 说明:

  - 行内代码并不一定要写代码,也可以作为着重标记

  ```markdown
  `这是一段行内代码`
  
  `<table border="1" cellspacing="0" width="500" height="500">`
  
  `print("Hello, World!")`
  
  `这是一行突出显示的文本内容`
  ```

  `这是一段行内代码`

  `<table border="1" cellspacing="0" width="500" height="500">`

  `print("Hello, World!")`

  `这是一行突出显示的文本内容`



### 7.2代码块

- 代码块的格式
  - \`\`\` + 语言
    代码
    \`\`\`
  - \~\~\~ + 语言
    代码
    \~\~\~
- 说明:
  - 在Typora中,可以输入<strong><kbd>Ctrl</kbd></strong> \+ <strong><kbd>Enter</kbd></strong>脱离代码块

## 11.拓展文本标记

MarkDown想实现更多文本显示效果,只能通过HTML

### 11.1键盘文本

- 键盘文本的格式:

  - \<kbd> + 文本内容 + \</kbd>

  ```markdown
  <kbd>文本内容</kbd>
  ```

  <kbd>文本内容</kbd>

### 11.2放大文本

- 放大文本的格式:

  - \<big> + 文本内容 + \</big>

  ```markdown
  <big>文本内容</big>
  ```

  <big>文本内容</big>

### 11.3缩小文本

- 缩小文本的格式:

  - \<small> + 文本内容 + \</small>

  ```markdown
  <small>文本内容</small>
  ```

  <small>文本内容</small>

### 11.4多彩文本

- 缩小文本的格式:

  - \<font color\=orange> + 文本内容 + \</font>(orange可以为其他颜色)

  ```markdown
  <font color=orange>文本内容</font>
  ```

  <small><font color=orange>文本内容</font></small>

## 12.拓展文本显示效果

拓展显示效果既不是原生 **Markdown语法** 支持的，也非 HTML标记，而是部分编辑器 提供的 **额外标识符**，属于拓展语法，旨在为 **Markdown使用者** 提供更多样式选择

不同编辑器不一样,此处以 **Typora** 为例

### 12.1文本高亮

- 文本高亮的格式:

  - <strong><kbd>\=\=</kbd></strong> + 文本内容 + <strong><kbd>\=\=</kbd></strong>

  ```markdown
  ==文本内容==
  ```

  ==文本内容==

### 12.2上标

- 上标的格式:

  - \^ + 文本内容 + \^

  ```markdown
  文本内容^文本内容^
  ```

  文本内容^文本内容^

### 12.3下标

- 下标的格式:

  - \~ + 文本内容 + \~

  ```markdown
  文本内容~文本内容~
  ```

  文本内容~文本内容~

### 12.4Emoji符号

- Emoji符号的格式:

  - \: + 表情文本 + \:

  ```markdown
  :smile:
  ```

  :smile:
